# RandomBlockBuild
랜덤블록 빌드배틀

## 소개
랜덤블록건축은 랜덤으로 주어진 9개의 블록만을 이용해서 1분안에 건축을 하는 콘텐츠입니다. 빌드배틀을 하거나 건축 퀴즈를 하는등 자유롭게 사용이 가능합니다.

## 나오지 않는 블록 리스트
- 화분 블록 (POTTED_*)
- 벽에 붙는 블록 (WALL_*)
- 간판 (SIGN)
- 매달린 간판 (HANGING_SIGN)
- 머리 (HEAD)
- 커맨드 블록 (COMMAND_BLOCK)
- 스트럭처 블록 (STRUCTURE_BLOCK)
- 지그소 블록 (JIGSAW)
- 엔드 포탈 (END_PORTAL)
- 네더 포탈 (NETHER_PORTAL)
- 빛 블록 (LIGHT)
- 움직이는 피스톤 (MOVING_PISTON)
- 개구리 알 (FROGSPAWN)
- 감염된 블록 (INFESTED)

## 게임 규칙
건축을 하는 사람은 자동으로 크리에이티브 모드가 되며, 랜덤으로 지급받은 블록 이외의 다른 블록은 사용이 불가능합니다. 제한시간이 종료되면 인벤토리가 초기화됩니다.
